## Fixes

### Issue 9 — LiveSessionPage event-ended cleanup
**File:** `client/src/features/live/LiveSessionPage.tsx`
**Change:** Replace the bare `setPhase('complete')` `useEffect` with the same teardown the `session:completed` socket handler does — clear LiveKit token, match, room id, dashboard, partner-disconnected, byeRound, transition status, all in one block. Idempotent (no-op if already in `complete`).

### Issue 10 — Background persistence
**Files:** `client/src/features/live/Lobby.tsx`, `client/src/features/live/VideoRoom.tsx`
**Change:**
- Persist `{ mode, imagePath }` in `sessionStorage` under key `rsn_bg_preference` whenever the user picks a background.
- On mount of both `LobbyMediaControls` (lobby) and `MediaControls` (breakout), once the local camera track is ready, read the saved preference and re-apply via `setProcessor`. Use the same `loadBgProcessors()` lazy-load path that already exists.
- Custom uploaded blob URLs survive same-tab navigation; on full reload they expire and revert to `disabled` cleanly.

### Issue 12 — Host tile adjacency in normal mode
**File:** `client/src/features/live/Lobby.tsx`
**Change:** Compute `actingHostCount = cameraTracks.filter(t => hostsSet.has(identity) && !tileDemotedSet.has(identity)).length`. When `actingHostCount >= 2`, host tiles render at `col-span-1 row-span-1` (normal size, ring kept for role signal) so they sit side-by-side. When `actingHostCount === 1`, keep the existing `col-span-2 row-span-2` big-tile treatment (preserves Phase Q director prominence).

### Issue 13 — Host self-demote (server + client)
**Files:** `server/src/services/orchestration/handlers/host-actions.ts`, `client/src/features/live/Lobby.tsx`
**Change:**
- **Server:** remove the director-self-demote guard at `host-actions.ts:2848-2853`. The director knows what they want. Visual-only; no privilege impact.
- **Client:** add a self-shrink/restore button on the director's own LOCAL tile when `isLocal && isHost && trackRef.participant.identity === hostUserId`. Mirrors the existing F2 button visibility logic for cohosts but with `isLocal` true. Visible only to the director themselves.
